# PRD — Помощник по изучению английского языка

## Обзор продукта

Интерактивный ассистент для самостоятельного изучения английского языка. Цель — помочь пользователю перейти с уровня A2 до B2 за 6–9 месяцев при ежедневных занятиях по 15 минут.

---

## Цель и задачи

- **Целевой пользователь:** человек с уровнем A2, который хочет достичь B2
- **Главная цель:** повысить уровень английского через короткие ежедневные занятия
- **Ключевое ограничение:** не более 15 минут в день

---

## Структура урока

Каждый урок состоит из трёх обязательных блоков:

| Блок | Назначение | Длительность |
|---|---|---|
| Приветствие | Настрой, мотивация, краткое введение в тему | 1–2 мин |
| Обучение | Правило / лексика / навык с примерами | 9–11 мин |
| Проверка | Интерактивный тест, самопроверка | 3–4 мин |

---

## Расписание по дням недели

Каждый день недели закреплён за отдельным навыком:

| День | Тема | Навык | Цвет |
|---|---|---|---|
| Понедельник | Времена глаголов | Грамматика | Фиолетовый |
| Вторник | Словарный запас | Лексика | Зелёный |
| Среда | Аудирование | Понимание на слух | Розовый |
| Четверг | Разговорные фразы | Речь | Янтарный |
| Пятница | Чтение | Чтение | Синий |
| Суббота | Письмо | Практика письма | Зелёный |
| Воскресенье | Повторение | Закрепление | Коралловый |

---

## Путь прогресса A2 → B2

Три фазы обучения:

**Фаза 1 — Фундамент (месяцы 1–3)**
- Базовые времена глаголов
- Частотная лексика (топ-500 слов)
- Простые тексты и диалоги

**Фаза 2 — Рост (месяцы 4–5)**
- Сложные грамматические конструкции
- Аудирование (native speed)
- Связное письмо

**Фаза 3 — Финиш (месяц 6)**
- Закрепление всего материала
- Симуляция реальных ситуаций
- Подготовка к уровню B2

---

## UI и дизайн

**Стиль:** дружелюбный, минималистичный, цветной

**Компоненты интерфейса:**
- Карточки дней недели — каждый день с уникальным цветом и иконкой
- Раскрывающиеся блоки (приветствие / обучение / проверка)
- Таймер обратного отсчёта (15 минут, становится красным при < 3 мин)
- Полоса прогресса A2 → B2 с милстоунами: A2 → A2+ → B1 → B1+ → B2
- Интерактивные тесты с полем ввода и автопроверкой
- Кнопки "Провести урок" и "Начать урок прямо сейчас" (запускают полный интерактивный урок)

**Цветовая система (по типу блока):**
- Приветствие → зелёный (#E1F5EE)
- Обучение → фиолетовый (#EEEDFE)
- Проверка → янтарный (#FAEEDA)

---

## Функциональность урока

Пример реализованного урока (Понедельник — Времена глаголов):

- Объяснение правила: Present Simple vs Present Continuous
- Таблица ключевых слов-подсказок (always / right now и т.д.)
- 5 интерактивных заданий с вводом ответа
- Автопроверка: зелёный фидбек при правильном ответе, красный + объяснение при ошибке
- Итоговый счёт с мотивирующим сообщением
- Кнопка перехода к следующему уроку

---

## Технический стек

- **Платформа:** Claude.ai (интерактивные виджеты)
- **Рендеринг:** HTML + CSS + vanilla JS внутри show_widget
- **AI-движок:** Anthropic API (claude-sonnet-4-6) для генерации уроков
- **Иконки:** Tabler Icons (outline)
- **Хранение состояния:** in-memory JS (без localStorage)

---

## Метрики успеха

- Пользователь занимается минимум 5 дней в неделю
- Прохождение теста проверки с результатом 4/5 и выше
- Достижение уровня B1 за первые 3 месяца
- Достижение уровня B2 за 6–9 месяцев
